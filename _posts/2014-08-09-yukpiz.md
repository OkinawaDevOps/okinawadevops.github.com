#### インストール

~~~sh
$ sudo apt-get update
$ sudo apt-get install docker.io
$ sudo ln -sf /usr/bin/docker.io /usr/local/bin/docker
~~~

**イメージを起動**

~~~sh
$ sudo docker run -it <image>
...
root@0e8d79b996570:/#
# <C-p>, <C-q>で起動したまま端末を抜ける
~~~
